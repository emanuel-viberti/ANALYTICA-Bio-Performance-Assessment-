**ANALYTICA** es una herramienta de diagnóstico nutricional avanzada diseñada para atletas de fuerza y usuarios de gimnasio. A diferencia de las calculadoras convencionales basadas en el IMC, esta plataforma utiliza el estándar científico **NHANES 2025** para estimar la masa muscular esquelética mediante la circunferencia del brazo ajustada.

## 🔬 Base Científica
La lógica de cálculo se basa en el estudio:
> *Costa-Pereira, J. P., et al. (2025). Arm circumference as a marker of muscle mass: cutoff values from NHANES 1999–2006. American Journal of Clinical Nutrition.*

La herramienta aplica factores de corrección biométricos para filtrar el tejido adiposo y obtener una métrica real de hipertrofia, permitiendo identificar puntos de corte críticos (28cm/26cm en hombres) para la salud y el rendimiento.

## 🚀 Funcionalidades Clave
- **Diagnóstico de Masa Muscular:** Basado en perímetros y corrección por IMC.
- **Dynamic Nutrient Timing:** Planificación de ingesta Pre y Post entreno según el horario real de entrenamiento.
- **Interfaz High-End:** Diseño optimizado para rendimiento (Dark Mode).
- **Conversión Directa:** Acceso a asesoría 1-a-1 personalizada.
